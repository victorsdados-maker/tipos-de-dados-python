# tipos-de-dados-python
# Exemplos de Tipos de Dados em Python

Este projeto demonstra o uso dos quatro tipos de dados básicos em Python:
- **str** (texto)
- **int** (números inteiros)
- **float** (números decimais)
- **bool** (valores booleanos: True ou False)

## Como usar
1. Execute o arquivo `tipos_dados.py` em Python 3.
2. O programa pedirá entradas para `str`, `int` e `float`.
3. Para `bool`, o exemplo já mostra valores fixos (`True` e `False`).

## Conceitos aplicados
- Entrada de dados com `input()`
- Conversão de tipos (`int()`, `float()`)
- Saída com `print()`
- Tipos básicos da linguagem Python
